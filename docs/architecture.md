## Objective

This lab demonstrates a basic Samba file server implementation for corporate environments.

The goal is to simulate a small enterprise setup including:

- Centralized file sharing
- User/group permissions
- Basic access control
- Operational documentation

## Components

- Linux Server (Ubuntu/Debian)
- Samba Server
- Local users and groups
- Shared directory with restricted access

## Features

- User-based authentication
- Group-based permissions
- Central shared folder
- Basic security configuration
- Automation via bash script

## Use Case

Designed as a learning lab for IT Support / Junior SysAdmin roles, focusing on:

- File services
- Identity basics
- Troubleshooting foundations
- Operational automation
